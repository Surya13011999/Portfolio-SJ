Track Go

Bus tracking is an application that tracks a bus and gathers the distance to each station along its route. This tracking System avoids the installation of an electronic device in a vehicle, and is replaced with an installed Android App on any Smartphone to enable the Administrator/User to track the vehicle's location. The tracking is enabled using the GPS in the android device that is on the bus. The users can get flexibility of planning travel using the app, to decide on which bus to take or when to catch the bus. The waiting time of the user can be reduced. The different queries and efficient route management can be easily done through central server system.
